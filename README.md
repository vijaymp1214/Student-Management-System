## 📖 Overview

The **Student Management System** is a desktop application built using **Java Swing** that allows users to manage student information through a graphical interface.

The project provides complete student record management with features like adding, updating, deleting, searching, saving, and loading records while maintaining a simple and user-friendly experience.

---

## ✨ Key Features

### 👨‍🎓 Student Management
- Add New Students
- Update Existing Records
- Delete Student Records
- Search Student Details
